# Furii Animation Studio

A training and employment platform for 2D/3D animators. Trainees apply, learn through structured lessons, submit real work, and grow into paid production roles.

## Tech Stack

- React 18 + TypeScript + Vite
- Supabase (auth, database, storage)
- Tailwind CSS + shadcn/ui
- Framer Motion
- TanStack Query

## Getting Started

```bash
npm install
npm run dev
```

Set the following environment variables:

```
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_anon_key
```
